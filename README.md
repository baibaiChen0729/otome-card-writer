**第一步**：打开你想用的 `.md` 文件，全选复制所有内容

## 🖤 角色卡的设计原则

这里的角色卡不只是「把角色设定列出来」——它们有自己的设计哲学：

**防 OOC 核心**
> 角色不说「我爱你」，但他用行动说了所有的话。
> 他不道歉，但他记住了你说过的每一件事。
> 他等你主动，等到了才说「这次是你自己来的」。

**性张力核心**
> 写意识，不写动作——他靠近你之前的那一步，比已经靠过来更有感觉。
> 写未完成——最有张力的场景是被打断的。
> 写沉默——他没说的话比他说出来的更重。

**结局设计**
> HE 是他终于做了那件他一直在克制的事。
> BE 是他没有拦你，他只是安静地承受了失去——他爱过你，但他也让你走了。
